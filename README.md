# Лабораторная работа 16

## Описание программы

Программа реализует работу с динамическими массивами вещественных чисел, выполняя два основных задания.

## Задание 1
Работа с математической функцией:
- Создание динамического массива заданного размера
- Заполнение массива значениями функции: `f(x) = 5x³ + 2x² - 15x - 6`, где x изменяется от 1.3 с шагом 0.1
- Округление всех элементов массива вверх с помощью `ceilf()`
- Вывод исходного и преобразованного массива
- Освобождение памяти

## Задание 2
Работа с массивом случайных чисел:
- Создание массива случайных чисел в диапазоне [-1.00, 1.00]
- Удаление элементов, значение которых превышает заданное A
- Поиск последнего отрицательного элемента
- Вставка значения -999 после найденного отрицательного элемента
- Вывод массивов на каждом этапе обработки

## Основные функции

1. **`full_elements()`** - заполнение массива значениями математической функции
2. **`calc_elements()`** - округление элементов массива вверх
3. **`put_elements()`** - вывод элементов массива
4. **`fill_random()`** - заполнение массива случайными числами
5. **`delete_k()`** - удаление элементов, больших заданного значения
6. **`find_last_negative()`** - поиск последнего отрицательного элемента
7. **`insert()`** - вставка элемента в массив
